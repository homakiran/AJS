1.RGB value:

syntax: rgb(red, green, blue)

Each parameter (red, green, and blue) defines the intensity of the color between 0 and 255.
For example, rgb(255, 0, 0) is displayed as red, because red is set to its highest value (255) and the others are set to 0.
To display the color black, all color parameters must be set to 0, like this: rgb(0, 0, 0).
To display the color white, all color parameters must be set to 255, like this: rgb(255, 255, 255).

2.HEX value:

syntax:#rrggbb (OR) #COMBINATION OF HEXADESINAMAL

Where rr (red), gg (green) and bb (blue) are hexadecimal values between 00 and ff (same as decimal 0-255).
For example, #ff0000 is displayed as red, because red is set to its highest value (ff) and the others are set to the lowest value (00)

3.HSL value:

syntax:hsl(hue, saturation, lightness)

Hue is a degree on the color wheel from "0 to 360". 0 is red, 120 is green, and 240 is blue.
Saturation is a "percentage value", 0% means a shade of gray, and 100% is the full color.
Lightness is also a "percentage", 0% is black, 50% is neither light or dark, 100% is white

4.RGBA value:

syntax:rgba(red, green, blue, alpha)

RGBA color values are an extension of RGB color values with an alpha channel - which specifies the opacity for a color.
The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all).

4.HSLA value:

syntax:hsla(hue, saturation, lightness, alpha)

HSLA color values are an extension of HSL color values with an alpha channel - which specifies the opacity for a color.
The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all).

EXAMPLE FOR TYPES OF COLORS REPRESENTATION IN HTML

Same as color name "Tomato":

#ff6347

hsl(9, 100%, 64%)

Same as color name "Tomato", but 50% transparent IS MENTIONED AS 0.5 IN CODE WE CAN CHANGE IT BETWEEN 0.0 to 1.O:

rgba(255, 99, 71, 0.5)

hsla(9, 100%, 64%, 0.5)

In addition to the predefined color names, colors can be specified using RGB, HEX, HSL, or even transparent colors using RGBA or HSLA color values.
